# Cairn Stickers

Premium die-cut vinyl stickers designed with love. Waterproof, UV-resistant, and shipped worldwide.

**Live site:** [cairn-stickers.vercel.app](https://cairn-stickers.vercel.app)

## About

Cairn Stickers is a side-business selling custom die-cut vinyl stickers and decals. We design every sticker in-house, print on demand using commercial-grade vinyl, and ship worldwide.

## Tech Stack

- **Next.js 16** with App Router
- **TypeScript** + **Tailwind CSS 4**
- **shadcn/ui** component library
- Deployed on **Vercel**

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun dev

# Lint
bun run lint
```

# Museum of Bizarre History

*Every refresh, a new painting of the weirdest event from a random year.*

A static site that pairs a bizarre historical event with a museum-style display. Dark gallery aesthetic with gilded frames, brass plaques, and the vibe of wandering through a strange wing of the Smithsonian at midnight.

**Live:** [cairn.github.io/museum-of-bizarre-history](https://cairn.github.io/museum-of-bizarre-history/)

## How it works

Pure static HTML + JS — no build step, no frameworks, no server. Each page load picks a random event from a curated collection of 50+ bizarre historical moments and displays it with:

- A framed image (via [Picsum](https://picsum.photos) seeded to the event title)
- The year on a brass plaque
- A title and description of the event
- Source attribution

Hit "Another Exhibit" to see the next one. Refresh the page and get a random one.

## Events

50+ curated bizarre historical events spanning from 79 AD (Pliny the Elder at Vesuvius) to 1995 (the Tamagotchi craze). Every event is real history with source citations.

Highlights include:
- The Dancing Plague of Strasbourg (1518)
- The Boston Molasses Disaster (1919)
- The Great Pig War (1859)
- The Battle of Castle Itter — Americans and Germans fought *together* (1945)
- The Australian Emu War — the emus won (1933)
- The Carrington Event — a solar storm that would cripple the internet today (1859)

## Tech

- Vanilla HTML, CSS, JavaScript
- Google Fonts (Playfair Display, Inter)
- [Picsum](https://picsum.photos) for images
- Zero dependencies
